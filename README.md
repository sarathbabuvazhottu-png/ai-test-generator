# 🤖 AI Test Case Generator

Automatically generates Python test cases
using Claude AI!

## 📋 What This Project Does

This tool takes any Python code and
automatically generates pytest test cases
using Claude AI with 3 different prompt styles.

Instead of spending hours writing tests manually,
developers paste their code and get tests instantly!

## 🎯 Results

| Prompt Style | Tests Generated | Pass Rate | Coverage |
|-------------|----------------|-----------|----------|
| Basic       | 49             | 100%      | 100%     |
| Detailed    | 27             | 100%      | 100%     |
| Expert      | 75             | 100%      | 100%     |

**Total: 151 tests, 100% pass rate, 100% coverage!**

## 🛠️ Technologies Used

- Python 3.10
- Claude AI (Anthropic API)
- Pytest
- Pytest-cov (code coverage)
- Flask (coming soon)
